*As a student. Didn't put that in the title.

You can get a static website like this one for free as long as you're a student! And in this blog, I'm going to tell you exactly how.
Some pre-requisites :

1) Very basic linux/cmd commands (Not mandatory but helpful, you can always search)
2) Basics of HTML to do some tweaks to our website
3) Understanding of Git and GitHub  
I wouldn't lie to you by saying you need zero technical knowledge to do this. But hey, it's super easy to learn and I'm providing all the resources you'll need. Plus you save a lot of money annually!

For the ones who just need a roadmap and can figure stuff out on their own via youtube or something, here's the quick version:
• GitHub pack -> NameCheap domain -> Install Ghost offline CLI and host              
• Download themes & customise -> Change .hbs files of Ghost theme to HTML & ROFF using WGET
• Use VS code or any editor to push these HTML files to your GitHub repo linked with the Namecheap account -> Get SSL, DNS and deploy https
• Edit with ghost and push files to make changes.

Step 1: GitHub Student Developer Pack

The GitHub student developer pack is a MUST for every college going student. It gives you access to premium packages of Canva, Microsoft Azure, JetBrains and 100s more. The one we'll be needing for making the blog is - Namecheap.

Sign up for the student developer pack by entering your details along with a picture of your college ID for verification. That's it.
Then head on to the benefits page and get student benefits. There you will find Namecheap's free .me domain name benefit. Link your Namecheap account with your GitHub account and get a GitHub repository with the domain name integrated.

Sign up here: GitHub Student Developer Pack - GitHub Education
NameCheap website: Namecheap Education Program: Free Domains for Students (nc.me)
Here's a youtube video to help - Free Domain and Web Hosting with GitHub Student Developer Pack and Namecheap - YouTube
And here's one to enforce https - Enforce HTTPS in GitHub Pages with Namecheap Domain - YouTube
Enforcing https will be done in Step 6 again so it's okay if you skip it for now.

Step 2: Get GHOST

Ghost is an open-source website builder. You can choose one of its amazing themes and customise offline all for free. To get the GHOST CLI offline follow their easy instructions: Ghost-CLI - A fully loaded tool for installation and configuration

Once how you have the Ghost CLI downloaded offline, here's how to launch it every time -
Open your command prompt from start menu
Type in just 2 lines of code - the ones I've underlined in red

It takes 10 secs to run and you'll have a link (underlined in blue), just copy-paste it in your browser and you'll have your ghost interface.

There are hundreds of things you can do with Ghost and we don't have the time for it all so I recommend you read the documentation by Ghost : Ghost: The #1 open source headless Node.js CMS

The most basic version is - you go to the theme tab and choose a theme. Then go to the pages tab and edit the 'About the Author' and similar pages (you can even remove them).
The Ghost UI is very friendly and has inbuilt guides to help you create your website easily.  The red underlined tabs in the UI image above are the tabs you'll usually use so explore those.

Go to the posts page and start writing your blog. Use all the easy to use integrated tools to add images and make it pretty with no code at all. Then you go to your theme and download it. (My theme is joelma).

Once you download the theme files you have them in .hbs file format.
Another thing you'd want to edit is your page footers and headers.

Open the theme footer and header files with VS code or any editor and edit the footer content. You don't particularly need HTML knowledge for this (but basics would help). You just need to find the text that you want to replace (like I've underlined with red in the image) and then replace it with your desired footer text and save it.

Notice how I've used container and row tags to change the layout of the footer (underlined in blue). If you're good with HTML CSS you'll be able to easily make a footer like this page's and integrate your social media links and everything there. If not then just ignore it, it isn't necessary. The heart of any blog is it's content not the design :)  

Step 3: WGET

Here's the thing. We're going to host our website on GitHub for free. GitHub only allows us to host a static website i.e no backend features. What do we mean by that? Try using the search button on this website, it won't work because it's a non static feature. It's nonetheless there because it would work when you host it on Ghost.org let's say (but you'll have to pay the yearly fee).

Static features are all you require as a student for a blog. At the end of the blog I've added the method to make a dynamic website for free as well, but I'd suggest you going for it if you're comfortable with static first.

GitHub websites should be made of .html files, whereas our themes are .hbs files.
An HBS file is a template file created by Handlebars, a web template system. It contains a template written in HTML code and embedded with Handlebars expressions. An HBS file performs the same function as a .HANDLEBARS file. We need to convert these into .html files before we can deploy them on GitHub.

Wget = Web Get. It crawls the websites and gets the HTML files we need for our GitHub page.
Download wget by following these instructions: How to Use wget Command With Examples | PhoenixNAP KB
Now,
1) Go to the Joelma/(your theme name folder) inside your editor and make a new folder inside it called docs. (very important)
2) Using your editor, open the terminal INSIDE THE JOELMA/(your theme) FOLDER.
Not inside docs. Not in C: drive. We need a terminal inside the theme folder and then enter the following command

wget -r -nH -P docs -E -T 2 -np -k (your localhost URL)

it should look something like this ->

Once you run this, your html files should be inside your docs folder!

Step 4: Push to GitHub

So here's what we have now.

-We got a domain name from namecheap + github student pack.
-We used a website making tool to help up avoid writing code (GHOST).
-We made the website but the https://localhost website that we run is offline, i.e. only we can see it because it's locally hosted.
-We want someone to host it online 24/7 (for which we usually pay) but GitHub was like hey you can host a static website here for free no problem.
-So we converted our handlebar .hbs (theme) files to .html for GitHub using wget.
-Now we send all of this to the GitHub repository which is integrated with our domain name and Github takes care of the rest.

Now sending files to GitHub is a process that you need to do EVERY TIME you make a new post. Like I'll have to do after I complete writing this. So what I personally suggest is you get familiar with github instead of copy pasting the commands every time, because it gets messy at times. You just need to learn the basics of add, commit, push, log, status, pull and branch.

We've to get the
SSL certificate : An SSL certificate is a digital certificate that authenticates a website's identity and enables an encrypted connection
DNS : special Domain Name System (DNS) record that allows domain owners to specify who is allowed to issue SSL certificates for their domains
And then move our websites to https so it's all safe and secure.

I would've loved to explain this but GitHub has done it and I can't do it better so

It's a small but important step, and with this your website is ready to roll!

Step 6: Editing and adding New Posts

Technically not a step but eh. So to edit what you have to do is :

1) Open ghost CLI using the commands
2) Use ghost to create a new post/edit posts
3) Download the theme again
4) Convert to .html using wget
5) Push to GitHub

Seems overwhelming I know. But once you do it once or twice then you'll barely take 10 mins after writing a new post to deploy it. It's a valuable skill to learn and definitely worth the time.

Every once in a while, we think of changing our lives around for the better. This change starts with a simple step, which, when done repeatedly, becomes a habit. So what is the best practice to cultivate to change our lives?

Brendon Burchard has been a high-performance coach for over a decade, and after a study in over 190 countries, he says
"Hands down, the habit that changed their trajectory was seeking clarity."
Reference:  High performance habits.

Seeking clarity is a habit? Yes, it is and here's why it's the most important habit. Clarity on the big questions of life is nothing short of a superpower. We have limited time and energy, and clarity helps you channel them both to that one thing that truly matters to you. If you fail to finish what you start, if you seek everyone's validation for everything, and if you can't say no to the things and people draining the life out of you, this is what you're missing.

People in the top 1% of their job category are always the people with clarity. If you ask them what five words best describe them, they'll fire it at you instantly. What are their values? Instant answers. When Opera Winfrey walks into a room for a meeting, her first question is, "What's our intention today?".

"Clarity precedes success." ~ Robin Sharma.

WHY EVEN BOTHER?

  1. We want people to understand us more than anything else. If we know who we are, we can connect with people better. If we can't understand ourselves, how can we expect others to? Know yourself
  2. Pareto's principle says: 80% of all your wealth, happiness or any desirable outcome depends on 20% of your input. Finding that 20% and channelizing our efforts towards it is the most crucial step of specialization. Find your 20.
  3. Life is just a series of choices. Every different choice leads to a different life. If you define your values as clear as water, all your preferences will be aligned. Clarity gives you the direction without which practising the wrong things would make you perfect at them.
  4. Jay Shetty explains how you're stressed when your mind is not aligned with your actions. You still do it when you're doing something, but you're unsure if it's right or when you hate your job or education. A clear image of what you want from your life in the next week or the next decade helps you align your actions with your thoughts. Eliminate stress.
  5. Most importantly, it gives you the strength to stand alone. When you dare to differ, you're going to doubt yourself every day when you see everyone else taking the other road. When you walk on a path where others don't, you need to know where you're going. Seek clarity.

OKAY, SO HOW DOES IT WORK?

The Bible and One republic (in the song counting stars) say, "seek it out, and ye shall find", and it is as simple as that.

We see ads every day on social media, on roads and almost everywhere. But can you recall the ads you see? We browse through Facebook to get updates on our friends and family, so we blur out whatever isn't about them. But remember that time when you were thinking of getting a new pair of shoes or headphones, and an ad shows up right on your feed, and you can't help but notice? Even when you're not looking for it, your brain's subconscious clarity on what you want makes you see that ad. When you define a crystal clear image of what you want from your life, even shortly, everything else blurs.

We've heard about the law of attraction, believing that positive thoughts attract positive things and people in life. But maybe we all have positive and negative things around us all the time, and only when we have the clarity of what we want and choose to focus on the positive do we stop seeing the negatives and see the positives everywhere!

HOW DO I SEEK CLARITY LIKE THE PROS?

Seeking clarity is an active process that requires you to put in the time to get the return. A high performer spends 74 more mins a week deliberately seeking clarity on different aspects of life than an average person.

You can go wrong while seeking clarity when you emphasize too much on the answers. Be obsessed with finding the right questions. When you have the right questions, you'll find the answers eventually. Clarity starts with the right questions.
Set apart some time daily or weekly to reflect on the work you do, how you interact with people, the good and bad habits, document everything. Write it down or record voice notes. Study yourself like you would study someone before hiring them.

Ask yourself :
1. Who am I? What are my strengths and weaknesses
2. What are my values in life?
3. What are my future aspirations? What everyday things that I do are steps towards it?
4. If I could describe my ideal self in the future, the person I'm trying to become, how would I describe them?  
5. Am I of service to people?

Questions like my future aspirations and what I want with life can be intimidating at first. Take small steps, list the things you like, the things you're good at, the items you want to be good at. Find an intersection; no matter how absurd it sounds, it's possible; it's the 21st century. Ignore the naysayers.

You can go as deep with the questions as your mind goes, and there's no right or wrong. It can be scary because you're going to be vulnerable, you're going to have the same experience you had when it was time to raise your hand in class to ask a doubt, but we know it's the only way to learn. The only thing to be scared of is ending up somewhere wrong because we were too busy or afraid to ask.

Clear thinking requires courage rather than intelligence. ~ Thomas Szasz

Ask yourself questions about the four essential aspects of life: Self, Social, Skill and Service. The 4-S of clarity.

Don't worry about getting all four perfectly at once. Every one of us has at least one of the four aspects under control, making it easier to have clarity. One by one, we put all four pieces together.

Remember that nowhere does it say the habit is getting clarity because getting to the end of it is not the goal; just actively seeking it does the work. It's about progress, not perfection.

THE WARREN BUFFET WAY

Take a notebook and list down 25 of the most important things you value in your life. Done? Now make sure you rank them in the order of importance. Then finally, scrape off the bottom 20 from the list. Yes, remove them from the list and your life at once.

You have to realize that the bottom 20 are draining your energy and time that you need to assign to the top 5 essential things in your life. These 20, although important, aren't important enough. That's why they're below the top 5. Things like social media have a bad reputation already. Anyone who wastes time on social media knows they are wasting their time. But these sneaky 20 things give you a sense of false productivity because they are essential. Still, they're taking away time from the top 5 things that could revolutionize your life, which is why they're so dangerous.

TAKE ACTION NOW.

Just thinking never works. Writing down what you came up with is the best way to solidify it in your head. Journalling is one of the best things you can do to seek clarity.

Clarity comes from action not thoughts ~ Marie Forleo

Once you have the clarity of values, of yourself and everything in between, compress it into words and expression.

• Find quotes that resonate.
• Write it down big and bold on a whiteboard
• Write it on a big piece of paper and stick it on the wall or a sticky note and stick it to your mirror.

Do whatever it takes to keep it in your sight. Get familiar with yourself. This is not a motivational youtube video that fires you up for a day and then long lost forever.

You can't be what you can't see. ~ Marian Wright Edelman

THE BIGGEST ENEMIES TO CLARITY

Asking the wrong questions can lead down the wrong path. Why don't I fit in? Why don't I have what he has? Why do I suck? What's wrong with me? These are the questions that make things worse. Avoid them.

"I like being spontaneous" is an excuse you might want to give yourself. But the ability to think and foresee is what makes us human. A balance of spontaneity and clarity makes life better.

You're never too busy for yourself. If you can't take out 60 mins in a week for navigating your ship of life to the direction you want to go in, you'll always be pushed around by the currents. You've made it to the end of the post, so I'm sure you will be a true seeker.

If you don't have time for the small things, you won't have time for the big things. ~Sir Richard Branson

Thanks for reading :)
